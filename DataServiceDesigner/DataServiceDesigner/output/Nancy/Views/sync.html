Manual synchronisation of user and role configuration does the following:

  • Updates the user and role configuration to match that of the external provider.
  • Updates configured permissions in the individual data services where required.

It should be used when users, roles or assignment of roles are changed in an external provider that does not automatically propagate changes to this system.


Synchronising the user and role configuration from the configured provider may take a significant amount of time.

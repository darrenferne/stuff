Edit Mode

When editing the parameters applied to the dependant view(s) the parameters bar is in 'Edit Mode'.

In this mode each parameter can be set with multiple values (or left empty to apply no restriction).

Once the parameters have been selected they can either be applied, or the changes discarded and the previously applied parameters will return.

Either action exits Edit Mode and updates the dependant view(s).

Refresh

The data shown in the dependant view(s) can be refreshed at any time by the user.

If updates to the data are live this should never be necessary.

Export

The current view is exported as in Excel format.

The currently selected parameters (if any) are applied and all pages are rendered in the exported file.

Edit View

This opens the editing pane for the setup of the current view, where changes can be made.

This functionality is identical to the normal view editing provided through the Explorer data service.

Pause/Resume Updates

Depending on the type of data being shown the view may be opened with either live data updates running or not.

Pause Updates

If the data is live then the updates can be paused.

Resume Updates

If the data is paused then the updates can be resumed to return it to a live state.

Show/Hide Help

Explorer includes built in help for much of its functionality.

Show Help

Shows the help buttons that can then be clicked to read the help for the appropriate functionality.

Hide Help

Hides the help buttons and closes the help pane if it is open.

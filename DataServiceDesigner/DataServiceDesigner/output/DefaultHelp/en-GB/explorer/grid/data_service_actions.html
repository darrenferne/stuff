Import

You can import data from other compatible systems into this instance of the current data service.

Visible to Role

Where data needs to have a visibility set it will be set to the role selected here.

It will not have the same access conditions set as where it was exported from.

Choose File...

This opens the file selection dialog so you can choose the file to import.

Export

You can export data that allows it from the current data service.

Type Selection

You can select which types to include in an export.

All data that your user can read will be included for each selected type will be included in the export.

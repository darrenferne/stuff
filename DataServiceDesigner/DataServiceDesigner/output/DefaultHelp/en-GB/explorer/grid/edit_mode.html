Edit Mode

Edit mode allows you to modify data directly in the grid.

The change tracking allows you to see what has been added or modified in the grid.

Yellow highlight indicates modified data cells.

Green highlight indicates inserted rows.

Keyboard Shortcuts

Key/Key Combination Action
Ctrl + A Select all
Ctrl + C Copy (when rows or cells are selected)
Ctrl + X Cut (will only copy and not remove uneditable cells)
Ctrl + V Paste (will not replace uneditable cells)
Alt + I Insert Row (when available)
Alt + R Remove Row (when available)
←/↑/→/↓ Move selection

Ctrl + Z

 Undo changes (single cell only)
Esc Unselect all
Tab Select next cell
Enter Select cell below or create new row (when in bottom row)
